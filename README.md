## 📖 简介

AI自动阅卷系统是一款基于大语言模型的智能批改工具，支持自动识别答题卡、AI评分、多服务商切换、历史记录导出等功能。

### ✨ 核心特性

- 🚀 **超紧凑界面** - 窗口面积减少45%，内容密度提升42%
- 🎨 **10种主题** - 实时切换，个性化体验
- ⚡ **高性能** - 支持批改几千份试卷，流畅不卡顿
- 🎯 **智能评分** - AI精准评分，36号超大分数显示
- 📊 **多格式导出** - 支持JSON、CSV、HTML、Word、Excel、PDF
- 🔄 **多服务商** - 支持10+AI服务商，一键切换

## 📖 使用指南

### 1. 配置答案
1. 切换到"答案配置"标签页
2. 填写年级、学科、题型
3. 输入参考答案和评分标准
4. 上传评分材料图片（可选）

### 2. 配置服务商
1. 切换到"服务商"标签页
2. 从下拉框选择或新增服务商
3. 配置API端点和密钥
4. 选择模型
5. 测试连接

### 3. 设置操作框
1. 切换到"操作框"标签页
2. 添加识别框、打分框、提交框
3. 点击"拖拽调整"可视化调整位置

### 4. 开始批改
1. 切换到"批改"标签页
2. 点击"🚀 开始批改"
3. 查看实时评分结果
4. 支持单次调试和连续批改

### 5. 查看历史
1. 切换到"历史"标签页
2. 选择导出格式（JSON/CSV/HTML等）
3. 点击"📦 导出"保存结果

### 6. 切换主题
1. 在底部状态栏找到🎨主题下拉框
2. 选择喜欢的主题
3. 立即生效，无需重启
